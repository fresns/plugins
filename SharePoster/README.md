- Installation with key name: `SharePoster`
- Installation using command: `php artisan market:require SharePoster`

## API

- Endpoint Path: `/api/share-poster/generate`
- Method: `GET`
- Request: `Query`

## Query Params

| Parameter Name | Type | Required | Description |
| --- | --- | --- | --- |
| type | String | **required** | `user`,`group`,`hashtag`,`post`,`comment` |
| fsid | String | **required** | uid or username, gid, hid, pid, cid |
| langTag | String | *optional* |  |
